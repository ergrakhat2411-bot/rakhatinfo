# 📑 Консольное портфолио «Обо мне» | Ергазин Рахат

Проект выполнен в рамках конкурса портфолио от **CapEducation** на треке Junior Python. 

Это интерактивное консольное приложение с текстовым меню, разработанное на объектно-ориентированном программировании (ООП). Программа знакомит пользователя с автором, его целями, процессом обучения и лучшими работами.

## 🛠 Особенности архитектуры
* **ООП-подход:** Логика инкапсулирована внутри класса `Portfolio`.
* **Разделение данных и логики:** Весь контент вынесен в файл `data.json`. Код не содержит захардкоженного текста («простыней» из print).
* **Бонусное оформление:** Реализованы аккуратные рамки, разделители и автоматическая очистка экрана консоли после каждого шага для удобства чтения.
